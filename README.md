# Chat Enhancement

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/tgarthoff-gh/chat-enhancement
   cd chat-enhancement
   ```
2. Open `index.html` in a web browser to test the application.

## Usage
- Use the file input to select multiple files (images, videos, documents).
- Click on the 'Upload' button to display the list of uploaded files.
